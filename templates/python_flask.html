Python Programming & Flask

Python is an object oriented programming language. Flask is a Python web framework that allows you to create web applications. It makes the creation of websites scalable.

Learning and Application of Knowledge

  • Assigning data to variables and using methods - use dir(object) to identify the variables that can be used

  • Using for loops, while loops and conditional if statements

  • String handling: slicing, splitting, stripping, etc.

  • Python collection tyes: tuples, lists, sets and dictionaries

  • Data storage and file handling
    - opening files and extracting specific information from them
    - lopping through a file to get a list of lines or words

  • Regular expressions are used to match or search for characters in a string (need to import the re module)

  • Functions allows you to group a block of code with repititive tasks together

  • Classes are used to group functionality together
    - usually a group of functions
    - the __init__() function is used to initialise the function

  • The Flask framework
    - allows you to create HTML templates to structure your webpage
    - makes use of HTTP methods like GET and POST;
    - GET is used to return data from the server (the default method)
    - POST is used to handle form data
    - planning to incorporate it properly in the next sprint if we had more time

Running dir(string) lists the methods you can use on a string
