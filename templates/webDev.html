Web Development

Web Development is all about websites and applications that are accessed through the internet and can run on a browser

All the websites and apps that are accessed through browsers such as Chrome are built by the technologies stated within this page.

How we applied our knowledge to the project:

  • HTML is used to apply the structure of the website.
  • CSS is used to style each page of a website.
  • JavaScript is used to add the functionality to the website

HTML was used as follows:

  • Headings are part of HTML and are used to give titles to different parts of a webpage such as the title and various other sections,
    and they range from "h1" through to "h6" where "h1" is the largest heading, "h4" is a heading the same size as the as a paragraph, and "h6" is the smallest size of a heading
  • Lists are also part of HTML and are a collection of items that can be either unordered, denoted as "ul" and bullet-pointed, or ordered, denoted as "ol" and numbered.
  • Images are added through the use of the "img" tag, and this allows for a image to be inserted from using a link from the internet, or using linking to a picture from the projects directory

CSS was used as follows:

  • Selectors are a vital component of CSS, and these allow for styling of the a collection of elements that are the same, or elements grouped within the same class
  • The "background-image" property is a property which let the background of the website be the image that we chose, instead of a regular color
  • The "width" and the "height" property lets you adjust the size of a specific element, whilst "margin" allows for adjustment of the position of an element on a webpage.
  • The "display" property, can be set to "relative" or "absolute", where "relative" means a element will move to places relative to its original place within the webpage, and "absolute" means that the element can move exactly where you define the element to move to

Javascript can be used as follows:

  • Variables are containers for data that can be used in different parts of a program, and they can store different types of data such as a string, number, and a boolean
  • Conditionals are blocks of code that execute a specific piece of code depending on a predefined condition being met, and these come in the form of a "if" statement
  • Loops are blocks that can repeat a portion of code a specified amount of times, and these are expressed in the form of a "for" or a "while" statement
  • Functions are a useful part of JS Programming that allows for a block of code to be repeated when a defined function has been called within the program
